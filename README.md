# DevConnect - Komunitas Programmer

Sebuah platform kolaboratif untuk para programmer Indonesia untuk belajar, berbagi pengetahuan, dan mengerjakan proyek open-source bersama.

## 🚀 Fitur Utama

- **Halaman Beranda** - Hero section dengan informasi komunitas
- **Showcase Proyek** - Pamerkan hasil karya terbaik dari komunitas
- **Daftar Anggota** - Lihat profil anggota tim dan peran mereka
- **Responsive Design** - Optimal di semua ukuran device
- **Modern UI** - Built dengan React, Tailwind CSS, dan Lucide Icons

## 🛠️ Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Package Manager**: npm

## 📦 Instalasi

```bash
# Clone repository
git clone https://github.com/aadimt/Tugas-Komunitas-Programmer.git

# Masuk ke direktori
cd Tugas-Komunitas-Programmer/app

# Install dependencies
npm install

# Jalankan development server
npm run dev
```

## 🔗 Akses Aplikasi

Aplikasi akan berjalan di `http://localhost:5173`

## 📝 Struktur Proyek

```
app/
├── src/
│   ├── js.jsx          # Main app component
│   ├── index.css       # Global styles
│   └── main.jsx        # Entry point
├── index.html          # HTML template
├── package.json        # Dependencies
├── vite.config.js      # Vite configuration
├── tailwind.config.js  # Tailwind configuration
└── postcss.config.js   # PostCSS configuration
```
